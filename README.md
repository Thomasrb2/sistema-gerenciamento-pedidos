# Sistema de gerenciamento de pedidos

## Sobre o projeto: 
Estou desenvolvendo este projeto com o foco de estar me aprofundando ainda mais em Java, ultilzando
a sua biblioteca, e o MySQL para estar guardando as informações fornecidas. Com o intuito de praticar
a integração entre linguagens.

## Tecnologias ultilizadas:
- Java
- JDBC
- Visual Studio Code
- MySQL

## Funcionalidades:
- Cadastro de clientes
- Cadastro de pedidos
- Alteração de informações
- Armazenamento de itens
